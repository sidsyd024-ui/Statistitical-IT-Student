     TYPES OF COMPUTERS
A) Supercomputer -an extremely fast powerful computer capable of performing trillions of calculations per socond
B) Mainframe -a large ,multiuser computer system designed for high volume data processing 
C) Minicomputer -more powerful than microcomputer but smaller than mainframe used in manufacturing plants 
D) Microcomputer -a small affordable computer intended for personal or office use
E)Laptop-a portable P.C. integrating all components into a compact, battery powered device suitable for mobile work 
F)Tablet-a touchscreen portable device used for browsing, reading, multimedia and productivity applications 
G) Smartphone -a handheld,mobile computer combining telecommunications, internet access,apps and sensors 
H) Embedded computer -a specialized systems integrated inside a large machine to perform dedicated functions e.g. car systems
